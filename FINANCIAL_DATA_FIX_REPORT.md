## 修复方案

### 1. 双接口策略
实现了优先级降级机制：
1. **优先使用**: `stock_financial_analysis_indicator`（直接返回财务指标）
2. **降级使用**: `stock_financial_report_sina`（从财务报表计算指标）

### 2. 财务报表计算逻辑
从新浪财经接口获取：
- **资产负债表**: 提取总资产、总负债、股东权益 → 计算资产负债率
- **利润表**: 提取净利润、营业收入、营业成本 → 计算 ROE、ROA、毛利率

计算公式：
```python
# 资产负债率 = 总负债 / 总资产 × 100%
debt_ratio = (total_liabilities / total_assets) * 100

# ROE（净资产收益率）= 净利润 / 股东权益 × 100%
roe = (net_profit / equity) * 100

# ROA（总资产收益率）= 净利润 / 总资产 × 100%
roa = (net_profit / total_assets) * 100

# 毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
gross_margin = ((revenue - operating_cost) / revenue) * 100
```

### 3. 缓存机制
- 添加 1 小时内存缓存，避免频繁请求
- 减少对数据源的压力
- 提高响应速度
