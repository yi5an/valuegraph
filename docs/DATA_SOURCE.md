# 数据源集成文档

**版本**：v1.0  
**更新日期**：2026-03-22

## 数据源概述

### AkShare（主要数据源）
- **类型**：免费开源
- **官网**：https://github.com/akfamily/akshare
- **优势**：
  - 完全免费，无需注册
  - 支持A股/美股/港股
  - 数据更新及时

### 数据源架构
```
DataSourceAdapter (抽象基类)
├── get_stock_list()       # 获取股票列表
├── get_financial_data()   # 获取财务数据
└── get_shareholders()     # 获取股东信息

实现类：
├── AkShareAdapter (A股)   ✅ 已实现
├── YahooFinanceAdapter    ⏸️ 待实现（美股）
└── TushareAdapter         ⏸️ 可选（付费）
```

## AkShare 接口说明

### 1. 股票列表
**接口**：`stock_zh_a_spot_em()`
**数据源**：东方财富
**返回字段**：
- 代码、名称、最新价、涨跌幅、成交量、成交额等

### 2. 财务数据
**接口 1**：`stock_financial_analysis_indicator(symbol)`
- 直接返回财务指标
- 优先使用

**接口 2**：`stock_financial_report_sina(stock, symbol)`
- 从财务报表计算指标
- 降级使用

**返回字段**：
- ROE（净资产收益率）
- ROA（总资产净利率）
- 毛利率
- 负债率
- 营收增长率

### 3. 股东信息
**接口**：`stock_gdfx_holding_analyse(symbol)`
**数据源**：东方财富
**返回字段**：
- 股东名称
- 持股数量
- 持股比例
- 变动情况

## 缓存策略

### Redis 缓存
- **股票列表**：缓存 1 小时
- **财务数据**：缓存 1 天
- **股东信息**：缓存 1 天

### 降级策略
1. 优先使用缓存数据
2. 缓存失效时调用 API
3. API 失败时返回历史数据
4. 所有数据失败时返回空数据

## 错误处理

### 常见错误
1. **网络超时**：重试 3 次，间隔 1s
2. **数据为空**：记录日志，返回空数据
3. **数据格式错误**：跳过该条数据，继续处理

### 日志记录
- 所有 API 调用记录到日志
- 错误信息包含：时间、接口、参数、错误详情
