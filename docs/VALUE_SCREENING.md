# 价值筛选算法说明

**版本**：v1.0  
**更新日期**：2026-03-22  
**理论基础**：巴菲特价值投资理念

## 筛选标准

### 1. 盈利能力（ROE ≥ 15%）
**指标**：净资产收益率（ROE）
**公式**：`ROE = 净利润 / 净资产`
**标准**：ROE ≥ 15%
**理由**：
- ROE 反映公司盈利能力
- 巴菲特标准：长期 ROE > 15%
- 高 ROE 意味着优秀的商业模式

### 2. 财务健康（负债率 ≤ 50%）
**指标**：资产负债率
**公式**：`负债率 = 总负债 / 总资产`
**标准**：负债率 ≤ 50%
**理由**：
- 低负债率意味着财务稳健
- 抗风险能力强
- 避免债务危机

### 3. 评分算法
**公式**：
```
Score = ROE × 0.5 + (100 - 负债率) × 0.5
```

**权重说明**：
- ROE 权重：50%
- 财务健康权重：50%

**示例**（贵州茅台）：
```
ROE = 24.32%
负债率 = 12.81%

Score = 24.32 × 0.5 + (100 - 12.81) × 0.5
      = 12.16 + 43.60
      = 55.76
```

## 筛选流程

```
1. 获取股票列表（≥100只）
   ↓
2. 获取每只股票的财务数据
   ↓
3. 应用筛选标准
   - 过滤：ROE < 15%
   - 过滤：负债率 > 50%
   ↓
4. 计算评分
   ↓
5. 按评分排序
   ↓
6. 返回 Top N
```

## 数据来源
- **ROE**：AkShare 财务数据接口
- **负债率**：AkShare 财务数据接口
- **更新频率**：每季度（跟随财报发布）

## 局限性
1. **历史数据**：仅基于历史财报，不保证未来表现
2. **行业差异**：不同行业标准不同（银行/保险需特殊处理）
3. **单一指标**：仅考虑 ROE 和负债率，未考虑其他因素
