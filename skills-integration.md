### 2️⃣ **Financial Analyst Skills**
**仓库**：https://github.com/alirezarezvani/claude-skills  
**定位**：财务分析与估值

**核心技能**：
- **财务比率分析**（ratio_calculator.py）

- **预测建模**（forecast_builder.py）
  - 驱动因素预测
  - 情景建模（基准/牛市/熊市）
  - 滚动预测

**5阶段工作流**：
```
Phase 1: Scoping（范围界定）
Phase 2: Data Analysis & Modeling（数据分析与建模）
Phase 3: Insight Generation（洞察生成）
Phase 4: Reporting（报告）
Phase 5: Follow-up（后续跟踪）
```

**适用场景**：
- 公司估值
- 财务健康评估
- 投资决策支持
- 预算管理

## 📋 综合分析工作流

### **Phase 1：市场环境分析**（TraderMonty）

**目的**：评估市场环境与时机

**步骤**：
1. **宏观环境检查**
   - 全球股指表现
   - 外汇、商品、债券
   - 市场情绪指标

2. **市场广度评估**
   - S&P 500广度指标
   - 牛熊市阶段判断
   - 市场健康评分

3. **板块轮动分析**
   - 周期vs防御板块
   - 板块相对强度
   - 市场周期阶段

4. **技术分析**
   - 周线图表分析
   - 趋势、支撑阻力
   - 情景概率评估

**输出**：
- 市场环境评分（0-100）
- 板块轮动建议
- 技术面判断

---

### **Phase 2：财务深度分析**（Octagon）

**目的**：评估公司财务质量

**步骤**：
1. **财务报表分析**
   - 利润表（营收、净利润、EPS）
   - 资产负债表（资产、负债、现金）
   - 现金流量表（OCF、FCF、净现金流）

2. **增长分析**
   - 营收YoY增长
   - 净利润YoY增长
   - 现金流YoY增长

3. **细分分析**
   - 产品细分（收入结构）
   - 地理细分（区域风险）

4. **财务健康评分**
   - Altman Z-Score（破产风险）
   - Piotroski Score（财务健康）
   - 历史评级趋势

5. **ESG评估**
   - MSCI评级
   - Sustainalytics风险
   - 行业排名

**输出**：
- 财务健康评分（0-100）
- 增长质量评估
- ESG风险等级

---

### **Phase 3：估值与投资决策**（Financial Analyst）

**目的**：计算内在价值与安全边际

**步骤**：
1. **财务比率分析**
   - 盈利能力（ROE、ROA、毛利率）
   - 流动性（流动比率、速动比率）
   - 杠杆（负债率、利息覆盖）
   - 效率（资产周转率）
   - 估值（P/E、P/B、EV/EBITDA）

2. **DCF估值**
   - WACC计算
   - 自由现金流预测
   - 终值计算
   - 敏感性分析

3. **预算差异分析**
   - 实际vs预期
   - 重要性识别
   - 趋势判断

4. **预测建模**
   - 驱动因素识别
   - 情景建模（基准/牛市/熊市）
   - 概率加权

**输出**：
- 内在价值区间
- 安全边际计算
- 情景分析报告

---

### **Phase 4：综合投资决策**

**整合三维度评分**：

| 维度 | 权重 | 评分（0-100） | 说明 |
|------|------|--------------|------|
| **市场环境** | 30% | - | 技术面+宏观环境 |
| **财务质量** | 40% | - | 财务健康+ESG |
| **估值吸引力** | 30% | - | DCF+安全边际 |

**投资决策矩阵**：

```
综合评分 = 市场环境×0.3 + 财务质量×0.4 + 估值×0.3

评分 ≥ 80：强烈买入
70-79：买入
60-69：持有
50-59：观望
< 50：避免/卖出
```

## 💡 实战应用：比亚迪案例

### **Phase 1：市场环境分析**

**宏观环境**：
- 全球电动车渗透率持续提升
- 中国新能源政策支持
- 特斯拉降价压力

**板块轮动**：
- 新能源板块：成长期
- 汽车板块：电动化转型

**技术分析**：
- 当前价：¥105.09
- 趋势：震荡上行
- 支撑：¥90
- 阻力：¥115-120

**市场环境评分**：**70/100**

---

### **Phase 2：财务质量分析**

**财务健康**：
- ROE：10.83%（偏低）
- 负债率：合理
- 现金流：正增长

**增长质量**：
- Q3营收：1949.85亿元（-3.05%）
- Q3净利润：78.23亿元（-32.60%）
- 但环比改善+23.07%

**ESG评级**：
- 新能源龙头，ESG表现良好

**财务质量评分**：**65/100**

---

### **Phase 3：估值分析**

**DCF估值**：
- WACC：10%
- 增长率：15%（保守）
- 内在价值：¥90-120

**安全边际**：
- 当前价：¥105.09
- 内在价值中位数：¥105
- 安全边际：0%（不足）

**估值评分**：**60/100**

---

### **Phase 4：综合决策**

**综合评分**：
```
= 70×0.3 + 65×0.4 + 60×0.3
= 21 + 26 + 18
= 65/100
```

**投资建议**：
- **持有**（60-69分）
- 不适合加仓（安全边际不足）
- 等待¥115-120减仓机会
