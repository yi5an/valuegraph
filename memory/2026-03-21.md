## 项目启动：ValueGraph - 价值投资知识图谱平台

### 产品定位
- **产品名称**：ValueGraph - 价值投资知识图谱平台
- **核心价值**：基于知识图谱的价值投资决策系统
- **目标用户**：个人投资者、投资新手、价值投资爱好者、专业投资者

### 核心功能模块（6大模块）
1. **多市场价值投资推荐**（P0）
   - 支持A股/美股/港股/印度等多市场
   - 基于ROE>15%、负债<50%、现金流为正等价值标准筛选
   - 数据源：Tushare Pro、Yahoo Finance、Alpha Vantage
   - 开发周期：4周

2. **财报深度分析**（P0）
   - 自动提取三大表（资产负债/利润/现金流）
   - 5-10年时间线展示财务数据
   - 财务健康度评分（ROE/ROA/毛利率等）
   - 开发周期：4周

3. **新闻资讯聚合**（P0）
   - 多源新闻实时聚合（新浪财经、雪球、Bloomberg等）
   - 重要人物推特监控（王传福、Elon Musk、Warren Buffett等）
   - 实时推送（Web/Telegram/邮件）
   - 开发周期：3周

4. **知识图谱构建**（P0）
   - 实体识别（公司/人物/股票/产业链）
   - 关系抽取（投资/竞争/合作/产业链）
   - 图谱可视化（Neo4j + G6）
   - 开发周期：5周

5. **图谱智能挖掘**（P1）
   - 产业链挖掘（上下游企业查询）
   - 关联风险分析（风险传导路径）
   - 智能问答（RAG + LLM）
   - 开发周期：4周

6. **持股信息查询**（P0，新增）
   - 公司股东结构（十大股东/机构持股）
   - 人物持仓查询（巴菲特/张磊等知名投资者）
   - 机构持仓查询（基金/券商/外资）
   - 持股变化监控（增持/减持实时提醒）
   - 数据源：Tushare Pro、SEC 13F、港交所披露易
   - 开发周期：3周

## 技术选型决策

### 为什么选择Neo4j而非其他图数据库？
- **成熟度高**：文档完善，社区活跃
- **查询语言**：Cypher语法简单易学
- **性能**：支持千万级节点，查询性能优秀
- **生态**：与Python集成良好

### 为什么选择FastAPI而非Django？
- **性能**：基于ASGI，性能优于Django
- **异步支持**：原生支持async/await
- **类型提示**：基于Pydantic，自动校验
- **开发效率**：自动生成OpenAPI文档

### 为什么选择Next.js而非纯React？
- **SSR支持**：SEO友好
- **API Routes**：前后端一体化
- **性能优化**：自动代码分割、图片优化
- **开发体验**：热更新、TypeScript支持

## 关键洞察

### 价值投资选股框架（五步法）
1. **财务健康度**：ROE>15%、负债<50%、现金流为正
2. **护城河分析**：品牌、成本、网络效应、转换成本、监管壁垒
3. **行业赛道**：市场规模大、增长可持续、竞争格局清晰
4. **管理层评估**：诚信、能力、股东友好、战略清晰
5. **估值与安全边际**：内在价值计算、安全边际>30%

### 持股信息的重要性
- **机构持股变化**是发现投资机会的重要信号
- **大股东增持/减持**往往预示股价走势
- **知名投资者持仓**（如巴菲特、张磊）是重要参考
- **产业链持股**可以发现上下游投资机会
