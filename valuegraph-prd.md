## 📋 产品概述

### 产品名称
**ValueGraph** - 价值投资知识图谱平台

### 产品定位
```
基于知识图谱的价值投资决策系统
帮助投资者从海量信息中挖掘投资机会
实时新闻聚合 + 深度图谱分析 + 智能推荐
```

### 目标用户
- **个人投资者**：想学习价值投资，但缺乏专业工具
- **投资新手**：不知道如何分析公司财务数据
- **价值投资爱好者**：想要系统化的选股工具
- **专业投资者**：需要深度分析工具

### 核心价值主张
```
1. 多市场价值投资推荐（A股/美股/港股/印度）
2. 深度财报分析 + 时间线展示
3. 实时新闻聚合 + 智能推送
4. 知识图谱深度挖掘
5. 持股信息实时监控
```

## 🎯 核心功能模块

### 模块1：多市场价值投资推荐 ⭐⭐⭐⭐⭐

#### 功能描述
支持用户选择不同市场（A股、美股、港股、印度等），基于价值投资理念推荐股票。

#### 详细需求

| 子功能 | 描述 | 优先级 |
|--------|------|--------|
| **市场切换** | 用户可选择A股/美股/港股/印度等市场 | P0 |
| **价值筛选器** | ROE>15%、负债<50%、现金流为正等标准 | P0 |
| **智能推荐** | 基于用户偏好+价值标准，推荐Top 10股票 | P1 |
| **自定义标准** | 用户可调整筛选参数（如ROE阈值） | P1 |
| **对比分析** | 多只股票横向对比 | P2 |

#### 技术挑战
```
✅ 优势：价值投资标准通用，跨市场适用
⚠️ 挑战：
- 不同市场财务报表格式不同（A股/美股/印度）
- 数据源差异大（需要对接多个API）
- 货币汇率实时转换
- 监管合规（不同市场法规不同）
```

#### 数据源方案

| 市场 | 数据源 | 成本 | 更新频率 |
|------|--------|------|---------|
| **A股** | Tushare Pro / 东方财富 | ¥500-2000/月 | 实时 |
| **美股** | Yahoo Finance / Alpha Vantage | $0-200/月 | 实时 |
| **港股** | 港交所 / 东方财富 | ¥500-1000/月 | 实时 |
| **印度** | NSE / BSE API | ₹1000-5000/月 | 实时 |

#### 开发周期
**4周**（包含数据对接+筛选算法+前端界面）

### 模块2：财报深度分析 ⭐⭐⭐⭐⭐

#### 功能描述
深度分析公司财报（年报/季报），按时间线展示财务数据变化趋势。

#### 详细需求

| 子功能 | 描述 | 优先级 |
|--------|------|--------|
| **财报解析** | 自动提取三大表（资产负债/利润/现金流） | P0 |
| **时间线视图** | 从近到远展示5-10年财务数据 | P0 |
| **趋势分析** | 自动识别增长/下滑趋势 | P0 |
| **财务健康度评分** | 基于ROE/负债/现金流等打分 | P1 |
| **异常检测** | 识别财务异常（如存货激增） | P1 |
| **同业对比** | 与行业平均水平对比 | P2 |

#### 核心指标

```
盈利能力：ROE、ROA、毛利率、净利率
偿债能力：负债率、流动比率、速动比率
成长能力：营收增长率、净利润增长率
运营能力：存货周转率、应收账款周转率
现金流：经营性现金流、自由现金流
```

#### 技术方案

```
数据采集：
- Python爬虫（Scrapy）+ OCR（解析PDF财报）
- 定时任务（每季度更新）

数据处理：
- 财务指标计算（ROE/ROA/毛利率等）
- 时间序列分析（趋势识别）
- 异常检测算法（Isolation Forest）

可视化：
- ECharts时间线图表
- 财务健康度雷达图
- 杜邦分析树状图
```

#### 开发周期
**4周**（包含爬虫+数据处理+可视化）

### 模块4：知识图谱构建 ⭐⭐⭐⭐⭐

#### 功能描述
从新闻、财报、公告中提取实体（公司/人物/股票/产业链），构建实体关系图谱。

#### 详细需求

| 子功能 | 描述 | 优先级 |
|--------|------|--------|
| **实体识别** | NER提取公司/人物/股票/产业链 | P0 |
| **关系抽取** | 识别实体间关系（投资/竞争/合作） | P0 |
| **图谱可视化** | 交互式图谱展示实体关系 | P0 |
| **路径查询** | 查询两个实体间的关联路径 | P1 |
| **影响力分析** | 计算实体在图谱中的影响力 | P1 |
| **社区发现** | 识别产业链/利益集团 | P2 |

#### 实体类型

```
公司实体：迈瑞医疗、宁德时代、比亚迪
人物实体：王传福、曾毓群、李西廷
股票实体：SZ300760、SZ300750、SZ002594
产业链：新能源产业链、半导体产业链、医药产业链
事件：并购、人事变动、政策发布
地点：中国、美国、欧洲
```

#### 关系类型

```
投资关系：红杉资本 -投资-> 字节跳动
竞争关系：比亚迪 -竞争-> 特斯拉
合作关系：宁德时代 -供应-> 特斯拉
产业链关系：宁德时代 -上游-> 赣锋锂业
人事关系：王传福 -CEO-> 比亚迪
地域关系：比亚迪 -位于-> 深圳
```

#### 技术方案

```
图数据库：Neo4j（成熟）/ ArangoDB（多模型）
查询语言：Cypher（Neo4j）/ AQL（ArangoDB）

图算法：
- PageRank：计算实体影响力
- Community Detection：发现产业链
- Shortest Path：关联路径查询
- Link Prediction：预测潜在关系

可视化：
- D3.js / ECharts Graph
- Cytoscape.js（专业图可视化）
- G6（蚂蚁金服图可视化引擎）
```

#### 开发周期
**5周**（包含NLP+图谱构建+可视化）

### 模块5：图谱智能挖掘 ⭐⭐⭐⭐

#### 功能描述
通过知识图谱深度挖掘公司、产业链、新闻、财报之间的隐藏关系。

#### 详细需求

| 子功能 | 描述 | 优先级 |
|--------|------|--------|
| **产业链挖掘** | 查询某公司的上下游企业 | P0 |
| **关联风险分析** | 分析风险传导路径 | P0 |
| **新闻关联分析** | 新闻事件与实体关联 | P1 |
| **智能问答** | 基于图谱回答投资问题 | P1 |
| **异常检测** | 发现异常关联（如利益输送） | P2 |

#### 典型应用场景

**场景1：产业链挖掘**
```
用户查询："宁德时代的上游供应商有哪些？"

图谱查询：
宁德时代 -[供应商]-> 赣锋锂业
宁德时代 -[供应商]-> 天齐锂业
宁德时代 -[供应商]-> 华友钴业

结果展示：
1. 赣锋锂业（锂矿）
2. 天齐锂业（锂矿）
3. 华友钴业（钴矿）
```

**场景2：关联风险挖掘**
```
用户查询："如果特斯拉股价下跌，会影响哪些A股公司？"

图谱推理：
特斯拉 -[客户]-> 宁德时代 -[供应商]-> 赣锋锂业
特斯拉 -[客户]-> 拓普集团

风险传导：
特斯拉↓ → 宁德时代↓ → 赣锋锂业↓
特斯拉↓ → 拓普集团↓
```

**场景3：新闻关联分析**
```
新闻："拜登宣布对中国电动车加征关税"

图谱分析：
影响实体：比亚迪、蔚来、小鹏
产业链：新能源汽车产业链
相关新闻：2024年欧盟反补贴调查
历史事件：2018年中美贸易战
```

#### 开发周期
**4周**（包含图算法+智能问答）

## ⚠️ 关键风险

### 技术风险
1. **数据质量**：财报数据清洗复杂
2. **NLP准确性**：中文实体识别准确率需>90%
3. **图谱规模**：千万级实体，查询性能优化
4. **实时性**：新闻爬取延迟<5分钟

### 合规风险
1. **数据版权**：财报/新闻数据版权问题
2. **荐股合规**：避免"荐股"嫌疑（强调"工具"而非"建议"）
3. **隐私保护**：用户数据安全
4. **跨境数据**：美股/印度数据跨境合规

### 市场风险
1. **竞争**：已有同花顺、东方财富等
2. **用户习惯**：用户可能不愿付费
3. **技术门槛**：需要全栈能力
